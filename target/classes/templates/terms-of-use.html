Правила использования

Дата последнего обновления: 03.06.2025

1. Общие положения

Настоящие правила использования определяют условия использования сервиса HiVPN.

2. Обязательства пользователей

Пользователи обязуются:

  • Использовать сервис только для легальных целей
  • Не распространять вредоносное ПО через сервис
  • Не использовать сервис для спама или массовой рассылки
  • Не нарушать права третьих лиц

3. Обязательства сервиса

HiVPN обязуется:

  • Обеспечивать стабильную работу сервиса
  • Защищать персональные данные пользователей
  • Предоставлять техническую поддержку
  • Соблюдать конфиденциальность передаваемых данных

4. Ответственность

Пользователь несет ответственность за:

  • Соблюдение правил использования
  • Безопасность своих учетных данных
  • Легальность использования сервиса

5. Изменения правил

Мы оставляем за собой право вносить изменения в данные правила использования. Все изменения будут опубликованы на этом сайте.
